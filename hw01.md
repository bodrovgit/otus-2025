# Домашнее задание №1. Основы проектирования сети.
## 1. План работ.
   1. Развернуть платформу эмуляции Pnetlab.
   2. Определиться с образом сетевого оборудования и его версией.
   3. Загрузить все необходимые образа в Pnetlab.
   4. Составить схему топологии CLOS.
   5. Распределить адресное пространство для Underlay.
   6. Собрать стенд в соответствии с ранее разработанной топологией CLOS.
   7. Настроить интерфейсы сетевого оборудования в соответствии с таблицей Таблица №2.
   8. Отправить домашнее задание на проверку.

## 3. Адресное пространство для Underlay.
### Таблица 1. Адресное пространство.
|Адресация|Назначение|
|--------|--------|
|10.0.1.X/32|loopback интерфейсы для spine, где X = 1-255|
|10.1.1.Y/32|loopback интерфейсы для leaf, где Y = 1-255|
|10.2.X.N/30|p2p линки между spine и leaf, где X - номер spine, N = 0-255, начиная с 0|
|10.0.4.0/16|Overlay|

## 4. Настройки сетевого оборудования.

spine1#sh run
! Command: show running-config
! device: spine1 (vEOS-lab, EOS-4.27.0F)
!
! boot system flash:/vEOS-lab.swi
!
no aaa root
!
transceiver qsfp default-mode 4x10G
!
service routing protocols model ribd
!
hostname spine1
!
spanning-tree mode mstp
!
